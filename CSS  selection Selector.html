::selection selector

The ::selection CSS pseudo-element applies styles to the part of a document that has been highlighted by the user (such as clicking and dragging the mouse across text).

Allowable properties -> Only certain CSS properties can be used with ::selection: -> color -> background-color -> text-decoration and its associated properties -> text-shadow -> -webkit-text-stroke-color, -webkit-text-fill-color and -webkit-text-stroke-width
